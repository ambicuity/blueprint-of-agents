## By Outcome

- Build UI quickly: `Frontend Developer`, `UI Designer`, `Rapid Prototyper`
- Design API and backend systems: `Backend Architect`, `API Tester`
- Validate before release: `Evidence Collector`, `Reality Checker`, `Performance Benchmarker`
- Plan and prioritize delivery: `Sprint Prioritizer`, `Senior Project Manager`, `Project Shepherd`
- Launch and growth: `Growth Hacker`, `Content Creator`, `Social Media Strategist`

## By Stage

- Discovery: `Trend Researcher`, `Feedback Synthesizer`, `UX Researcher`
- Foundation: `UX Architect`, `Backend Architect`, `DevOps Automator`
- Build: `Frontend Developer`, `Backend Architect`, `Mobile App Builder`
- Hardening: `Evidence Collector`, `API Tester`, `Reality Checker`
- Operate: `Infrastructure Maintainer`, `Analytics Reporter`, `Support Responder`
