## If you want value in under 10 minutes

1. Pick a workflow from [workflows/index.md](../workflows/index.md).
2. Copy one prompt and run it in your AI coding assistant.
3. Compare output against the expected artifacts in that workflow.

## If you want one specialist agent

- Use [choose-an-agent.md](choose-an-agent.md).
- Then open the selected blueprint and copy its activation prompt.

## If you want orchestration

- Read [playbooks/QUICKSTART.md](../playbooks/QUICKSTART.md).
- Then run [playbooks/orchestration-framework.md](../playbooks/orchestration-framework.md).

## If you want to contribute

- Start with [CONTRIBUTING.md](../CONTRIBUTING.md#first-pr-in-15-minutes).
